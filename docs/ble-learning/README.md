# BLE 学习文档索引

这个目录用于长期维护 BLE 学习资料。文档应按编号阅读，从基础概念逐步进入 ESP-IDF BLE 开发、调试和优化。

## 推荐顺序

1. `00-学习路线.md`：学习目标、阶段划分、实验清单。
2. `01-BLE基础概念.md`：BLE、GAP、GATT、Service、Characteristic、UUID。
3. `02-广播与扫描.md`：广播包、扫描响应、设备发现流程。
4. `03-GATT服务与特征.md`：服务设计、读写、通知、订阅。
5. `04-ESP-IDF-BLE工程结构.md`：项目结构、初始化流程、配置项。
6. `05-连接参数与稳定性.md`：连接间隔、超时、MTU、重连策略。
7. `06-安全配对与绑定.md`：配对、绑定、密钥保存和隐私。
8. `07-低功耗与性能优化.md`：功耗指标、广播间隔、连接参数取舍。
9. `08-调试方法与常见问题.md`：日志、抓包、手机工具、问题定位。

## 维护规则

- 新文档使用两位数字前缀，保证阅读顺序稳定。
- 每篇文档应包含“目标”“核心概念”“ESP-IDF 关联点”“实践任务”。
- 当代码新增 BLE 功能或出现重要调试结论时，同步更新对应文档。
- 若学习顺序调整，先更新本索引，再移动或重命名具体文档。
